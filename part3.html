    Sequence

    A sequence is set of tasks in order, which are executed one after the other. Any number of tasks can be added but none can be skipped. An example of a sequence is a flow chart.


    Selection

    A selection is also called a decision. A question is asked and the computer makes a decision based on the answer of the question.

    An example of a selection is an if/else statement. If the answer is W, then X is done, else, if the answer is Y, Z is done.

    If(number == 3){ number + 1; }Else{ Return number; }

    Iteration

    The computer goes through a group of tasks and does something with the tasks, or it continues for ever.
    An example is a While loop

    While(number <10){ Number - 1; }
