Part 1

Describe the application and limits of procedural, Object Orientated and event driven paradigms.

Procedural Programming

Where the program is broken down into procedures which are steps.

You need to create a sequence of these procedures to make the program compile.

These tell what the computer does first and then second etc:

Examples of these include:
  • C
  • FORTRAN
  • And Pascal
    • Applications

    Applications include scientific/engineering calculations.

    Limitations

    The limitations of procedual programming is that it's harder to maintain as the codebase gets more complicated. Also, to correct errors, you need to go through all the program.

    Object Oriantated

    Overview

    Object oriantated languages (OOP) uses objects to organise the code.

    These objects make its easier to work with the code.

    Examples of these include:

  • Java

  • Python
  • Javascript
    • Applications
    Easier to change code, classes can be created and called on, this helps to split up the code. Such applications of OOP are large projects, such as web apps.
    Limitations

    Bigger file size, therefore the server has to be bigger to support the bigger file servers. They also need more memory to run the programs.
